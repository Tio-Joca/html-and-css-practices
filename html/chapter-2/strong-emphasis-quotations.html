Strong, emphasis and quotations

In this webpage, I start to manipulate some semantic markup provided by HTML, these elements are strong, em, blockquote and q. Semantic markup elements are used to provide special meaning to its contents, this type of element is useful for tools like search engines and screen readers. Right down this paraghaph you can see some of these elements in action.

When you look too much for an abyss, the abyss can destroy your self-esteem
