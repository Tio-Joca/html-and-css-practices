Images and its positions

Here, I will demonstraste the usage of images and its interaction with paragraphs. Firstly, we need to discuss about block and inline elements.

Block elements are HTML elements that always appear on a next line (e.g. paragraphs, ordered lists, list item). Inline elements are HTML elements that sit within block level elements and doesn't appear on a next line (e.g. images, links, line breaks).

If you want to see more examples about block and inline elements, you'll find here.

But, which correlation exists between paragraphs and images elements?

Image elements that sit outside paragraphs can be displayed before or after paragraphs, based on image element position in relation about a paragraph element. If an image element is located before a paragraph element, the respective paragraphs will be displayed on a new line. If an image element is located after a paragraph element, the respective image will be displayed on a new line.

Image elements that sit within paragraphs change which way its respective images content and paragraphs content are displayed, based where its image elements are located, in the beginning, in the middle or in the end of paragraphs element content.
