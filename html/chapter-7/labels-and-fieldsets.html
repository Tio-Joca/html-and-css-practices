Labels and fieldsets

Welcome back to another web page practice in this project. Now, I'll show to you how we can use labels and fieldsets in HTML. Labels are used in HTML to give a simple name for each form control that can be used as a clickable area. This element helps users making their interactions with the form controls more intuitive and could also be useful by vision impaired users. Fieldsets are used in HTML to separate form controls in sections. It should have a legend element to contain a caption that describes what this fieldset is related to.
