Radio buttons and checkboxes

In this page, I'll show you how to make radio buttons and checkboxes inputs. Radio buttons are used when a user needs to select one option from a short set of options. Checkboxes are used when a user needs to select or deselect one or more options from a short set of options.

To make radio buttons, we need to use an input element with a type attribute with value radio. Also, to make checkboxes, we need to use an input element with a type attribute with value checkbox.

Differently from text inputs, inputs where users uses their mices to select an option to be sent to a web server, contains a value determined by default for each option. It's important to say, when we look at these kind of input, name attribute and its respective value will be the same for each option.

Radio buttons and checkboxes have a value attribute to contain a data specified to be sent to the web server. Drop down list and multiple select boxes also use this attribute for the same purpose.

Also, we can use a checked attribute with radio buttons or checkboxes. The checked attribute is used to select an option by default. If it is used with radio buttons, only one checked attribute can be used with only one input element with type attribute with radio as its value and name attribute with a same value to its related set of options. If a checked attribute is used with checkboxes, we can use more than one checked attribute with input elements whose type attribute is checkbox and name attribute value is the same to its related set of options.
