Drop down list and multiple select box

Continuing this practice project, in this page I'll demonstrated the usage of drop down lists and multiple select boxes. To implement drop down lists, we need to use select element and option elements. Select elements are used to make drop down list and multiple select box, this element needs a name attribute to indicate the name that will be used to create the name/value pairs. In addition, we can use a multiple attribute to create multiple select boxes. Also, if we want, a size attribute can be used with drop down list boxes and multiple select boxes. This attribute can adjust the size of multiple select boxes. But, when we use this attribute without a multiple attribute, we can create a select box that allow users to select only one option in the related set of options.
