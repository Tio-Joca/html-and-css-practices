Image buttons, buttons and hidden controls

Welcome to another web page practice. Continuing our travel through forms in HTML, now I'll talk about image buttons, buttons and hidden controls. Image buttons are similar to submit buttons, the difference resides in the fact that image buttons use images to make an area entirely clickable to serve as a button. Buttons are HTML elements that can be used to make customizable buttons with images, text and other contents desired by the web author. Hidden form controls are used to maintain certain kinds of data the users doesn't need know about it, but these data is useful for some purpose in the web page or for a web server.
